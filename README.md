# FutureBlog

Cloudflare Pagesで公開するためのシンプルでモダンな静的ブログテンプレートです。

## 特徴
- **高速**: 純粋なHTML/CSS/JSで構築されているため、読み込みが非常に速いです。
- **デザイン**: グラスモルフィズムを採用したモダンなダークテーマ。
- **レスポンシブ**: スマートフォン、タブレット、PCに対応。
- **SEO最適化**: 基本的なメタタグを完備。

## デプロイ方法
1. このリポジトリをGitHubにプッシュします。
2. Cloudflare Pagesの管理画面から「Create a project」を選択。
3. GitHubリポジトリを連携。
4. ビルド設定（特になし、そのまま）でデプロイ。

## 記事の追加
`posts/` ディレクトリに新しいHTMLファイルを作成し、`index.html` からリンクを追加してください。
